API

You can interface with Yucando programmatically through our REST API. The documentation is shown below. You will need to authenticate each request using either a username:password combination, or you will need to attach a TOKEN *jwt* to your request.

Your jwt is:

Endpoints

GET /task/id/

Return task with specified Mongo ID

GET /task/points/

Return all tasks with given number of points

GET /task/project/

Return all tasks in a given project name, where project name is a string

GET /task/time_min/

Return all tasks with durations longer than time_min in seconds

GET /task/time_max/

Return all tasks with durations shorter than time_max in seconds

DELETE /task/

Delete all tasks with Mongo ID

Deferred methods

PUT /task/

Create a task with task name

GET /task

Return all tasks as JSON array

POST /task

Create a new task, return created record in JSON array
